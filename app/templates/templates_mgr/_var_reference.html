可用變數說明
{# ── 內建變數 ── #}
內建自動注入 每次寄信時自動帶入，無需設定
{{ '{{ date }}' }} 今日日期 YYYY-MM-DD
{{ '{{ datetime }}' }} 寄信時間 YYYY-MM-DD HH:MM
{{ '{{ year }}' }} 年份（如 2026）
{{ '{{ month }}' }} 月份（如 4）
{{ '{{ day }}' }} 日期（如 11）
{# ── 爬蟲動態變數 ── #}
爬蟲動態變數 執行時即時擷取注入
在下方「Scraper 變數」區塊綁定爬蟲後，
執行時自動擷取並注入對應變數。
含換行時加 filter：
{{ '{{ var | nl2br }}' }}
